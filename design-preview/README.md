# 微服务知识管理器 - UI 设计预览

## 设计理念

### Refined Industrial Aesthetic
精致的工业主义美学，深受 IDE（如 VS Code、JetBrains 系列）设计语言启发。

### 核心特点
- **深邃的背景**：#0d1117 深邃黑，提供专注的工作环境
- **电光蓝强调色**：#3b82f6，类似代码高亮的蓝色
- **非对称布局**：打破传统的左侧菜单+右侧内容布局
- **微妙的动画**：页面加载时的错落揭示动画
- **精致的阴影**：深色环境中的微妙光影效果

## 页面结构

### 1. 仪表盘（Dashboard）
- 欢迎信息
- 统计卡片：设计空间、服务知识、历史设计、个人模板
- 最近活动列表

### 2. 服务知识管理（Service Knowledge）
- 服务列表卡片
- 服务状态：已发布/草稿
- 快速操作：添加服务、查看详情

### 3. 设计空间管理（Design Space）
- 设计空间卡片
- Git 状态显示
- 快速操作：创建空间、打开空间

## 交互设计

### 动画效果
- **页面加载**：fadeInUp 错落揭示动画
- **卡片悬停**：translateY(-4px) 上浮效果 + 发光阴影
- **按钮悬停**：发光效果

### 状态反馈
- Git 状态：已连接（绿色）/ 待提交（黄色）
- 服务状态：已发布（绿色）/ 草稿（黄色）
- 悬停反馈：边框颜色变化 + 阴影增强

## 如何预览

## 设计决策

### 为什么选择工业主义？
1. **符合用户身份**：架构师/技术负责人熟悉 IDE 风格的界面
2. **避免沉闷**：传统的企业后台过于严肃，工业主义带来专业感和活力
3. **专注体验**：深色背景减少视觉干扰，提高工作专注度

### 为什么非对称布局？
1. **打破常规**：传统的左侧菜单布局千篇一律
2. **提升辨识度**：非对称设计让人印象深刻
3. **空间利用**：更好地利用屏幕空间展示内容

### 为什么使用错落动画？
1. **视觉引导**：错落揭示引导用户视线
2. **层次感**：创造视觉层次，而不是同时展示所有内容
3. **精致感**：微妙的动画细节体现设计用心
